Questions

  1. What kind of photos are good to encode using JPEG?

    Photos that are good to encode using JPEG are images that have a variety of colors, and photos that are used for sharing online.

  2. What kind of photos are good to encode using GIF?

    These are the kinds of photos that have simple graphics, animations, web elements, and images with limited colors.

  3. What is unique about the PNG format?

    This type of file uses lossless compression, which means that no image data is lost during compression. This can show clearer photos on websites, and anywhere that there is a media reader software.

  4. How do you resize your photo to the size it should be displayed?

    There are different kinds of ways to do this, but the easiest way to do so is by using width and height; manipulating both.

  5. Why should you resize photos using photo editing software instead of resizing it using CSS?

    Resizing photos with editing software keeps the image quality high and reduces the file size, which helps the website load faster.
