What are different ways to style your form?

HTML forms can be styled using inline CSS, internal styles within a style tag, or by linking to an external CSS file for better organization. Frameworks like Bootstrap offer pre-built classes for form elements, making responsive and consistent styling easy. Tailwind CSS provides utility classes for customized form styling without writing extensive custom CSS. CSS variables can also help manage theme colors or styles across the form by defining reusable values. JavaScript can be used for dynamic style changes, such as highlighting fields on focus or showing validation feedback.

Why does the professor like to use flex?

Because it provides an efficient way to design flexible, responsive layouts that adapt well across different screen sizes. Flexbox simplifies the alignment, spacing, and distribution of elements in a container.
