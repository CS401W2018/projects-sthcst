What you learned about Flex and CSS?

Flex is a powerful tool that allows you to create complex layouts with ease. It is a great way to create responsive designs that work on all devices.

Why do you need to have an accessible website? What does that mean?

Having an accessible website means that everyone can access your content, regardless of their abilities. This is important because it ensures that your website is inclusive and can be used by everyone.
