Internet

Web

This is where all documents and resources are formatted using Hypertext Markup Language, or HTML.

How are the Internet and Web different?

A network that connects all computers and devices around the world.

ASCII and Unicode

American Standard Code for Information Interchange is the encoding standard of an electronic communcation. This is mostly used in computers, devices, and other technologies to generate text.

What is UTF-8?

This is a format that translates Unicode characters to binary strings, and vice versa.

URL

Uniform Resource Locator is used to locate a specific resource in the internet. Parts of a URL includes the protocol and domain name.

Domain Name

These are used to identify services provided through the internet such as websites, and more.

HTTP

Hypertext Transfer Protocol is a set of rules that helps computers communicate through the internet.

HTTPS

Hypertext Transfer Protocol Secure is a secure version of the HTTPS which is also a set of rules that helps computers communicates through the internet securely.

IP Address

A string of characters that identifies each computer using the Internet Protocol to communicate over a network.

DNS

Domain Name System is a system that translates human language domain names to a language that computers understand.

W3C Recommendations

This is a specific set of guidelines that has received endorsement of W3C Members.

XML, XHTML, HTML, HTML5

These are the different versions of HTML that are very much needed in determining different meanings behind documents. Extensible Markup Language are the rules for the construction of the documents.
