What is Javascript? How did it get its name?

JavaScript is a programming language that was created by Brendan Eich in 1995. It was originally called Mocha, then LiveScript, and finally JavaScript. The name JavaScript was chosen to capitalize on the popularity of Java at the time.

What are the two ways you can run JavasScript?

There are two ways to run JavaScript: in the browser and on the server. In the browser, JavaScript is executed by the browser's JavaScript engine. On the server, JavaScript is executed by a server-side JavaScript engine like Node.js.

What was the most popular front-end js library?

The most popular front-end JavaScript library is jQuery. jQuery is a fast, small, and feature-rich JavaScript library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.

What is the most popular front-end js library today?

The most popular front-end JavaScript library today is React. React is a JavaScript library for building user interfaces. It is maintained by Facebook and a community of individual developers and companies.
